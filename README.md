# 天津大学视觉形象识别系统

为更便捷、规范地使用我校标识，本项目在天津大学视觉形象识别手册的基础上，进行了图形、文字的校正工作，并整理了完整的使用版本。

> [!NOTE]
> 本项目非天津大学官方项目，如有问题请以官方为准。图形标志和字体标志的版权归天津大学所有。
